# Contributing to LocalForge

Thanks for your interest! Here's how to contribute:

## Pull Request Process

1. **Fork the repo** and create a feature branch (`feat/your-feature`)
2. **Write code** following existing patterns (TypeScript, same style)
3. **Ensure it compiles**: `npx tsc --noEmit`
4. **Open a PR** against the `master` branch
5. **Wait for review** — the maintainer (Nrupal) must approve before merging

## Branch Protection

- `master` is protected — no direct commits
- All PRs require at least 1 approving review
- CI must pass (GitHub Actions: build on Node 20 & 22)

## Reporting Issues

Use the [issue templates](.github/ISSUE_TEMPLATE/) — they'll be emailed to the maintainer.

## Questions?

Open a [GitHub Discussion](https://github.com/nrupala/LocalForge/discussions).
